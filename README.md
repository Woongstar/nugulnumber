# Nugu랑 숫자놀이
## SKT와 함께하는 AI인재양성 사업
### 음성기반 AI (NUGU)로 하는 숫자놀이 

* AWS EC2 를 이용한 서버 초기 세팅
* Python3 Flask를 이용한 Api서버 계발
* NUGU DEVELOPER로 SKT NUGU서비스 음성 놀이 서비스 개발
* 팀 누굴넘버 PM/기획/개발
* VUX 플로우 설계

* 누구랑 숫자놀이
* SKT NUGU 음성인식을 기반으로한 어린아이들을 위한 두뇌훈련 숫자놀이
* 상세 PPT참고
* SKT사장상/최우수상 수상(2020/12/21)
